# 💰 가계부 웹 앱

개인 수입과 지출을 체계적으로 관리할 수 있는 웹 기반 가계부 프로그램입니다.

## 🔗 바로 사용하기

**👉 [https://dashing-crostata-0352a5.netlify.app](https://dashing-crostata-0352a5.netlify.app)**

---

## ✨ 주요 기능

- **📋 내역 관리** — 수입/지출 입력, 수정, 삭제
- **🔍 필터 조회** — 유형·카테고리·월별 필터링
- **📊 통계** — 월별 수입·지출·잔액 요약
- **📈 그래프** — 월별 막대 그래프, 카테고리별 도넛 차트, 요일별 지출 패턴
- **🤖 AI 조언** — 소비 패턴 분석 및 맞춤형 재정 조언
- **📥 CSV 내보내기** — 전체 내역 엑셀 파일로 저장
- **💾 데이터 유지** — 브라우저 localStorage에 저장 (껐다 켜도 유지)

---

## 🚀 설치 없이 바로 사용

별도 설치 없이 브라우저에서 바로 사용 가능합니다.

### 로컬에서 실행하기

```bash
# 저장소 클론
git clone https://github.com/your-username/가계부.git
cd 가계부

# 브라우저로 열기 (AI 기능 사용하려면 로컬 서버 필요)
python -m http.server 8000
# → http://localhost:8000 접속
```

> ⚠️ `index.html`을 파일로 직접 열면 AI 조언 기능이 작동하지 않습니다.
> 로컬 서버를 통해 열거나 배포된 링크를 사용하세요.
